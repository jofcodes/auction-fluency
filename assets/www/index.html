App Information

Auction Fluency v1.2.0

Self-paced onboarding for Meta ads auction system. 5 days, ~15 min per day, designed for Portal touchscreen landscape use. Built with Android WebView shell — no server backend, fully offline capable for quizzes.

Package: com.meta.auctionfluency  |  Target API 28  |  Built with shell aapt javac d8 apksigner (no Gradle)

Learning Science Basis

This app applies evidence-based learning principles: active recall via quizzes immediately after concepts (testing effect), spaced exposure across 5 days, elaborative interrogation via explain-back flashcards, interleaving of multiple question types, and immediate corrective feedback with explanations to strengthen memory traces and reduce illusion of fluency.
